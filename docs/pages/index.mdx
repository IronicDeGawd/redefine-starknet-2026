# ZKCred

Privacy-preserving credentials for Bitcoin holders on Starknet.

## What is ZKCred?

ZKCred enables Bitcoin holders to create verifiable credentials that prove their holdings tier without revealing their wallet address or exact balance. It's the bridge between Bitcoin wealth and on-chain identity.

:::info
**Built for RE{DEFINE} Hackathon 2026** by the Starknet Foundation.
:::

## Key Features

- **Privacy-First**: Wallet addresses are never stored on-chain, only hashed pubkeys
- **Oracle-Verified**: Balance verification through trusted oracles (Mempool.space)
- **On-Chain Storage**: Credentials stored on Starknet for trustless verification
- **Public API**: Easy integration for third-party applications
