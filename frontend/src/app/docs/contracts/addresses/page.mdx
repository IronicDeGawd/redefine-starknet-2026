## Interacting with Contracts

### Using starknet.js

```typescript
import { Contract, RpcProvider } from 'starknet';

const provider = new RpcProvider({
  nodeUrl: 'https://starknet-sepolia.public.blastapi.io',
});

const registry = new Contract(
  REGISTRY_ABI,
  '0x073566a1fcb9f4f7ca7a0f8d0e056929c59ccf435b8ae88ca1a39251bfeed216',
  provider
);

// Read credential
const credential = await registry.get_credential(credentialId);
console.log('Tier:', credential.tier);
```

### Using starkli (CLI)

```bash
# Read credential
starkli call \
  0x073566a1fcb9f4f7ca7a0f8d0e056929c59ccf435b8ae88ca1a39251bfeed216 \
  get_credential \
  0x123...  # credential_id

# Verify tier
starkli call \
  0x073566a1fcb9f4f7ca7a0f8d0e056929c59ccf435b8ae88ca1a39251bfeed216 \
  verify_tier \
  0x123... \  # credential_id
  2           # min_tier (Fish)
```
