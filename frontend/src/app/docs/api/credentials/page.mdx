# Credentials Endpoint

Retrieve and verify individual credentials.

## Get Credential

Retrieve details about a specific credential.

```
GET /api/v1/credentials/{id}
```

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string | Yes | The credential ID (hex string starting with `0x`) |

### Response

```json
{
  "id": "0x073566a1fcb9f4f7ca7a0f8d0e056929c59ccf435b8ae88ca1a39251bfeed216",
  "type": "btc_tier",
  "tier": 2,
  "tierName": "Fish",
  "issuedAt": "2026-02-24T12:00:00.000Z",
  "revoked": false,
  "status": "active"
}
```

### Example

```bash
curl -X GET \
  'https://api.zkcred.xyz/v1/credentials/0x073566...' \
  -H 'X-API-Key: zkcred_test_demo123'
```

## Verify Credential

Verify a credential is valid, optionally checking minimum tier.

```
POST /api/v1/credentials/{id}/verify
```

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string | Yes | The credential ID |

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `minTier` | number | No | Minimum tier required (0-3) |

### Response

```json
{
  "valid": true,
  "credential": {
    "id": "0x073566...",
    "type": "btc_tier",
    "tier": 2,
    "tierName": "Fish",
    "issuedAt": "2026-02-24T12:00:00.000Z",
    "status": "active"
  },
  "meetsTier": true
}
```

### Validation Failures

**Credential not found:**
```json
{
  "valid": false,
  "reason": "not_found"
}
```

**Credential revoked:**
```json
{
  "valid": false,
  "reason": "revoked",
  "credential": { ... }
}
```

**Below minimum tier:**
```json
{
  "valid": false,
  "reason": "below_tier",
  "credential": { ... },
  "meetsTier": false
}
```

### Example

```bash
# Verify credential meets Whale tier (3)
curl -X POST \
  'https://api.zkcred.xyz/v1/credentials/0x073566.../verify' \
  -H 'X-API-Key: zkcred_test_demo123' \
  -H 'Content-Type: application/json' \
  -d '{"minTier": 3}'
```

## Use Cases

### Access Control

```typescript
async function canAccessWhaleLounge(credentialId: string): Promise<boolean> {
  const response = await fetch(
    `${API_URL}/credentials/${credentialId}/verify`,
    {
      method: 'POST',
      headers: {
        'X-API-Key': API_KEY,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ minTier: 3 }),
    }
  );

  const result = await response.json();
  return result.valid && result.meetsTier;
}
```

### Display Tier Badge

```typescript
async function getUserTier(credentialId: string): Promise<string> {
  const response = await fetch(
    `${API_URL}/credentials/${credentialId}`,
    {
      headers: { 'X-API-Key': API_KEY },
    }
  );

  const credential = await response.json();
  return credential.tierName; // "Whale", "Fish", etc.
}
```
